# 응급의학과 담벼락 🏥

응급의학과 전문의들이 과별 협진 정보, 프로토콜, 공지사항을 실시간으로 공유하는 게시판입니다.  
Firebase Firestore 기반 실시간 저장 · 동기화 / PWA 지원으로 앱처럼 사용 가능합니다.

---

## 📋 주요 기능

| 기능 | 설명 |
|------|------|
| **⭐ 중요 탭** | 중요 표시한 글만 모아보기. 글에서 ⭐ 버튼으로 토글 |
| **🏥 응급의학과 탭** | 과 내부 공지·정보 전용 고정 탭 |
| **🗓 스케줄 탭** | 당직표 등 이미지 갤러리. 클릭하면 크게 보기 |
| **📖 사용방법 탭** | 홈 화면 추가 방법 · 기능 안내 (아코디언 UI) |
| **과별 탭** | 정형외과, 신경외과, 내과 등 탭 전환 |
| **탭 추가 / 수정 / 삭제** | `+ 과 추가` 버튼 · 탭의 ✎ 버튼으로 이름 변경 및 삭제 |
| **게시물 유형** | 정보공유 / 프로토콜 / 공지 / 일반 |
| **수정 / 삭제** | 게시물 하단 ✏️ 수정, 🗑 삭제 버튼 |
| **이미지 자동 압축** | 스케줄 탭 이미지 업로드 시 자동 압축 (Base64 저장) |
| **실시간 동기화** | 누군가 글을 올리면 모든 접속자 화면에 즉시 반영 |
| **PWA 지원** | 홈 화면에 추가하면 앱처럼 전체 화면으로 실행 |

## 🚀 배포 방법 (처음 한 번만)

### 1단계 — GitHub에 파일 올리기

저장소에 아래 4개 파일을 모두 업로드하세요.

```
index.html
manifest.json
icon-192.png
icon-512.png
```

### 2단계 — GitHub Pages 켜기

1. 저장소 **Settings** 탭 클릭
2. 왼쪽 메뉴 **Pages** 클릭
3. Branch: **main**, 폴더: **/ (root)** → **Save**
4. 1~2분 후 아래 주소로 접속 가능

```
https://hyinuyu.github.io/er-board/
```

## 🔥 Firebase 설정 방법

### 1. Firebase 프로젝트 만들기

1. [console.firebase.google.com](https://console.firebase.google.com) 접속
2. **프로젝트 추가** → 이름 입력 → Google Analytics 사용 안함 → 완료

### 2. Firestore 데이터베이스 만들기

1. 왼쪽 메뉴 **Firestore Database** 클릭
2. **데이터베이스 만들기** → Standard 버전
3. **테스트 모드**로 시작
4. 위치: **asia-northeast3 (Seoul)** 선택 → 완료

### 3. 설정값 확인

- 프로젝트 설정(⚙️) → **`</>`** 웹 앱 등록 → `firebaseConfig` 값 확인

> 이 프로젝트의 Firebase 설정값은 이미 `index.html`에 포함되어 있습니다.

## 🔒 보안 규칙 (30일 후)

Firebase Console → Firestore → **규칙** 탭에서 아래로 교체하세요.

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
```

## 📱 홈 화면에 추가하는 방법

### 아이폰 (Safari)
1. **Safari**로 접속
2. 하단 **□↑ 공유 버튼** 탭
3. **홈 화면에 추가** 선택
4. **추가** 탭 → 완료

### 안드로이드 (Chrome)
1. **Chrome**으로 접속
2. 우측 상단 **⋮ 메뉴** 탭
3. **홈 화면에 추가** 또는 **앱 설치** 선택
4. **설치** 탭 → 완료

> 앱 내 **사용방법 탭**에도 자세히 안내되어 있습니다.

## 🛠 문제 해결

**"연결 오류" 메시지가 뜰 때**
- Firebase Console → Firestore → 규칙 탭에서 읽기/쓰기 허용 확인
- 테스트 모드 30일 만료 여부 확인

**이미지 업로드가 안 될 때**
- 이미지가 너무 크면 압축 후에도 저장이 안 될 수 있어요
- 더 작은 해상도의 이미지를 사용하거나 미리 압축해서 올려주세요
